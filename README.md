# 自助售货机管理系统

基于 Linux 环境开发的轻量级自动售货机管理系统，使用链表数据结构实现商品交易、库存管理与权限管控一体化。

## 功能特性

### 顾客模式

- 浏览商品列表（仅显示可用商品）
- 购买商品（支持数量选择、支付找零）
- 实时库存更新

### 管理员模式

- 管理员登录（密码验证，3次尝试机会）
- 查看所有商品
- 添加/删除/修改商品信息
- 商品补货
- 查看交易记录
- 查看统计信息（商品总数、交易总数、总收入、总库存）

## 技术架构

### 数据结构

- **商品链表**：存储商品信息（编号、名称、价格、库存、可用性）
- **交易记录链表**：记录每笔交易（商品ID、名称、价格、数量、时间）

### 文件模块

| 文件                | 功能                             |
| ------------------- | -------------------------------- |
| `main.c`            | 主程序入口、菜单系统、系统初始化 |
| `vending_machine.h` | 头文件，定义结构体和函数声明     |
| `llist.c / llist.h` | 链表数据结构实现                 |
| `product.c`         | 商品管理（增删改查）             |
| `transaction.c`     | 交易记录管理                     |
| `inventory.c`       | 库存管理                         |
| `admin.c`           | 管理员功能模块                   |
| `customer.c`        | 顾客功能模块                     |
| `storage.c`         | 数据持久化存储                   |

## 编译与运行

### 编译

```bash

```

### 运行

```bash

```


## 数据持久化

系统会自动保存以下数据到文件：

- `products.dat` - 商品数据
- `transactions.dat` - 交易记录
- `system.dat` - 系统配置（总收入）

退出系统时自动保存，启动时自动加载。

## 项目特点

1. **轻量级设计**：纯C语言实现，无需外部依赖
2. **权限管控**：区分管理员和顾客角色
3. **数据持久化**：支持数据的保存与加载
4. **链表实现**：高效的动态数据管理
5. **跨平台兼容**：支持 Windows 和 Linux 平台
